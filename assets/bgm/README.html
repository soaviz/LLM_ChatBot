index.html BGM_PLAYLIST 등록 방법
const BGM_PLAYLIST = [
  // 기본 — 음악 파일만
  { title: '곡 이름',  file: 'assets/bgm/main/song01.mp3' },

  // 앨범 자켓 있을 때 — cover 필드 추가
  { title: '곡 이름',  file: 'assets/bgm/main/song02.mp3',
    cover: 'assets/bgm/covers/cover02.jpg' },
];
cover 필드를 추가하면 플레이어 앨범아트에 자동으로 이미지가 표시됩니다.
없으면 기본 그라데이션 아트로 표시됩니다.
파일 추가 순서
  • 1
    mp3 파일을 assets/bgm/main/ 에 넣기
  • 2
    앨범 이미지를 assets/bgm/covers/ 에 넣기 (선택)
  • 3
    index.html 열고 BGM_PLAYLIST 배열에 위 형식으로 등록
  • 4
    저장 후 git push — 앱에서 바로 재생 확인
